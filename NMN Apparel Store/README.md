# To run the App

1. Start the Backend Server
2. Start the Frontend

## To run the Admin App

Start 2 different terminals.

### In first terminal,

    cd Backend
    npm start

The above commands will start the Backend Server on port 2000

### In second terminal,

    cd Frontend
    cd admin-app
    npm start

The above commands will start the Front end App

## To run the User App

Start 2 different terminals.

### In first terminal,

    cd Backend
    npm start

The above commands will start the Backend Server on port 2000

### In second terminal,

    cd Frontend
    cd user-app
    npm start

The above commands will start the Front end App
